PAYMENT INTEGRATION
==================================================

Current Status: Ready for integration

Stripe:
1. Get API keys from https://dashboard.stripe.com
2. Add to .env:
   STRIPE_PUBLIC_KEY=pk_test_xxx
   STRIPE_SECRET_KEY=sk_test_xxx
3. Update donation handler

MTN MoMo:
1. Register at https://momoapi.mtn.com/
2. Get credentials
3. Add to .env:
   MTN_MOMO_API_KEY=xxx
   MTN_MOMO_API_URL=xxx
4. Update momo payment handler

==================================================

Images not showing?
‚Üí Check file paths in HTML
‚Üí Images must be in /images/ folder

API not working?
‚Üí Check server is running: npm start
‚Üí Check browser console for errors

Database problems?
